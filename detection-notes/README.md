# Detection Notes

This section records detection ideas, investigation notes, and observations made during log analysis exercises.

## What This Includes
- Initial alert hypotheses
- Indicators reviewed during investigations
- False positive considerations
- Detection improvement ideas

## Example Detection Thinking
- Identify unusual authentication patterns
- Review command execution frequency
- Correlate multiple events across time windows
- Assess whether behaviour is benign or malicious

## False Positive Awareness
- Consider legitimate admin activity
- Account for scheduled tasks and automation
- Validate alerts against business context

## Value
- Demonstrates analytical thinking
- Shows understanding beyond tool usage
- Reflects real-world SOC investigation workflow
